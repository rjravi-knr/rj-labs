
# Scope: Platform Application

This is the Unified Platform / Control Center for RJ-Labs. It serves as the central hub for managing tenants, configurations, and core operational operational tasks.

## Key Features
- Centralized Dashboard
- Auth Configuration Management
- Tenant Management
