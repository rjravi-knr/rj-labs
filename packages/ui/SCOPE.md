# Scope: UI Component Library

## Overview
`@labs/ui` is the shared component library for RJ-Labs. It provides a consistent design system and reusable components for all applications in the monorepo.

> [!IMPORTANT]
> This package contains **Pure UI** components only. It should **not** contain business logic, API calls, or app-specific state.

- Button
- Card
- Code

## Usage
Import components directly from the package:
```tsx
import { Button } from "@labs/ui/button";
```
