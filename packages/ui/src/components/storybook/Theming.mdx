# Theming

The UI library uses CSS variables for theming, allowing for easy customization and dark mode support. These variables are defined in `src/styles/globals.css`.

## Color Palette

The color palette is semantic, meaning colors are named after their function rather than their hue.

```css
:root {
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;

  --card: 0 0% 100%;
  --card-foreground: 222.2 84% 4.9%;

  --popover: 0 0% 100%;
  --popover-foreground: 222.2 84% 4.9%;

  --primary: 222.2 47.4% 11.2%;
  --primary-foreground: 210 40% 98%;

  --secondary: 210 40% 96.1%;
  --secondary-foreground: 222.2 47.4% 11.2%;

  --muted: 210 40% 96.1%;
  --muted-foreground: 215.4 16.3% 46.9%;

  --accent: 210 40% 96.1%;

  --input: 214.3 31.8% 91.4%;

  --radius: 0.5rem;
```

## Customizing

To customize the theme, you can override these CSS variables in your application's global CSS file.

### Example: Changing the Primary Color

To change the primary color to blue, update the `--primary` and `--primary-foreground` variables:

```css
:root {
  --primary: 221.2 83.2% 53.3%; /* Blue 600 */
  --primary-foreground: 210 40% 98%;
}
```

## Typography

The library uses standard Tailwind CSS font utilities. You can customize the font family in your `tailwind.config.js`.

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      fontFamily: {
        sans: ['Inter', 'sans-serif'],
      },
    },
  },
}
```
