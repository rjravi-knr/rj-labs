# Installation

To use the UI components in your project, install the `@workspace/ui` package:

```sh
pnpm add @workspace/ui
```

or with npm:

```sh
npm install @workspace/ui
```

or with yarn:

```sh
yarn add @workspace/ui
```

Then import the components you need:

```tsx
import { Card, Button } from '@workspace/ui';
```
